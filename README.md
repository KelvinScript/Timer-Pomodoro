# 🍅 Timer Pomodoro

Um Cronômetro Pomodoro interativo desenvolvido com HTML, CSS e JavaScript.

## 📖 Sobre o Projeto

O **Timer Pomodoro** é uma aplicação web construída para auxiliar na produtividade e no gerenciamento de tempo, baseada na mundialmente conhecida [Técnica Pomodoro](https://pt.wikipedia.org/wiki/T%C3%A9cnica_pomodoro). A ferramenta consiste em um cronômetro que divide o fluxo de trabalho em blocos de foco (tradicionalmente de 25 minutos), seguidos por curtos períodos de descanso.

Este projeto foi desenvolvido como parte do meu portfólio pessoal para consolidar conhecimentos práticos em desenvolvimento Front-end. O foco principal durante a construção foi aplicar a lógica de programação em JavaScript para manipular o DOM (Document Object Model), gerenciar eventos de tempo (utilizando `setInterval` e `clearInterval`) e criar uma interface de usuário limpa, responsiva e focada na usabilidade.

## ✨ Funcionalidades

- **Cronômetro de Foco e Pausa:** Alternância entre os ciclos de trabalho e os ciclos de descanso.
- **Controles Interativos:** Botões intuitivos para **Iniciar**, **Pausar** e **Zerar** a contagem a qualquer momento.
- **Feedback Visual:** Atualização do tempo restante em tempo real na tela.
- **Interface Minimalista:** Design planejado para evitar distrações e manter o foco do usuário na tarefa atual.
